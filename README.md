# Lapo Hub X — Scripts

Scripts feitos com a [Lapo Library X](https://github.com/LapoLapoNaldo/Lapo-X).

- **LapoHubX.lua** — Hub principal (stats, quests, limit break, banners, stages, traits, skins, webhook)
- **LapoSkillsX.lua** — Hub de habilidades (auto skills, starkei, sabor inf dmg, grave event, inf damage farm, trocar unit meio da partida)
- **LapoBoatX.lua** — BABFT (Player: voo, noclip, air walk, pulo infinito, walkspeed/jump | Farm: autofarm de gold com walk on air, run time e gold ganho)

Carregam a library localmente primeiro (via `Library.lua`) com fallback automático para o GitHub.
